## *Codigos Importantes*

> -  ####  *Mandar al cliente a lo que desea ver*

```
<body>
        <header>

            <h1>Perfumeria 🤑</h1>

            <!--Sirve para guiar al usuario al tema que desea en vez de bajar
			hasta el tema-->
			
            <nav>
                <!--Los navs son mas que todo para la navegacion interna de la
				pagina-->
				
                <a class=link href="#catálogo">Catálogo de Perfumes</a>
                <a class=link href="#oferpromos">Ofertas y Promociones</a>
                <a class=link href="#noved">Novedades / Lanzamientos</a>
                <a class=link href="#recomen">Más Vendidos / 
				Recomendados</a>
                <a class=link href="#contacto">Contacto</a>
                <a class=link href="#test">Test de Fragancias</a>

            </nav>

        </header>
>/body>
		

```
> - ####*Llevar al cliente al carrito de compras*

```
<nav>
                    
                    <!--Tambien se puede hacer navegacion con links externos-->
                    <!--Lo que se hace aca es ir al link creando una pagina nueva sin eliminarla-->
                    <!--target = pagina de explorador, _blank = nueva -->
                    <!--noreferrer se refiere a un problema de seguridad haciendo que no se muestre informacion de la pagina por la que se vino-->
                    <a 

                    href="carritocompras.html"
                    target="_blank"
                    
                    rel="noreferrer"

                    >Carrito Compras</a>
                
                </nav>
```
